# dotfiles
Quick castling by running `install.sh`

## Requirements
#### default shell: zsh
If your default shell is not zsh, please install zsh first.
1. Install via apt,
    ```sh
    apt install -y zsh
    ```
    or via brew
    ```sh
    brew install zsh
    ```
2. Change default shell
    ```sh
    chsh -s $(which zsh)
    ```
3. Logout and login again
